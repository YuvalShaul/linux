#### Linux

- The [Linux Kernel](https://www.kernel.org/) is where the kernel is coming from
- [The GNU Project](https://www.gnu.org/home.en.html)  is where The "Core" Programs are coming from:  
The vast majority of the "basic" commands you expect in a terminal come from the Free Software Foundation (FSF) via the GNU Project. This is why many people, including Richard Stallman, insist on calling the OS GNU/Linux.

- GNU Coreutils: This is where you get ls, cp, mv, rm, and cat.
- GNU Bash: The actual shell interface you type into.
- GNU Toolchain: This includes GCC (the compiler), GDB (the debugger), and glibc (the standard C library that almost all programs use to talk to the kernel).
